Theory Reference

Step 1 — log(w)
Circles → lines. Droste image
→ doubly periodic grid:
  vertical period: 2π (rotation)
  horizontal period: logN (zoom)

Step 2 — α · log(w)
Rotate + scale so a diagonal
path becomes a vertical segment
of height exactly 2π.
α must be complex — imaginary part
couples scale to rotation.

Step 3 — exp(α · log(w))
Vertical height-2π line → circle.
The loop closes. No hole.

α = (2πi + logN) / (2πi)
  = 1 − i·logN/(2π)
γ = exp(2πi·logN/(2πi+logN))
|γ| ≈ 22.584   arg ≈ 157.6°
